# BeamNG Web Dashboard
This simple mod and server can turn any device into a dashboard for BeamNG drive.  

# Setup  
Download the release zip.
## Part 1: BeamNG Drive Mod  
### Step 1.1:  
Move the notdbrandSim.zip file into the BeamNG Drive mod folder.  
### Step 1.2:  
In Beamng Drive, go to the options page, then select the 'other' sub menu. Then in 'Protocols', enable 'Other protocols'.  
  
## Part 2: Server  
### Requirements:  
- Server needs to run on the same computer as BeamNG Drive (Unless you modify the script).
- **Python 3**
- Libraries: Flask
  ```bash
  pip install flask
  ```
### Step 2.1:  
Extract the server and its files from the zip and install the requirements.    
### Step 2.2:  
Start the server by running 'server.py'.  

# Connecting
Use the IP address of the hosting computer and the port 5000 (Default) in a web browser to view the dashboard.

# Tested:  
OS: Windows 11  
BeamNG Drive: v0.38.5.0
